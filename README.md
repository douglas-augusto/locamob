## Setup

1 - git clone https://github.com/douglas-augusto/locamob.git
2 - rename .env.example for .env
3 - composer install
4 - php artisan key:generate
5 - php artisan migrate:install
6 - php artisan migrate

Versão do PHP = 7.4
Versão do Laravel = 6

Enjoy!
